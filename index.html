What is Cydia ?

Cydia is a Package Manager for iOS which enables the installation of software packages on jailbroken iPhones, iPads and iPod touch.

//

Cydia Development

Cydia provides a graphical user interface to jailbroken users using Advanced Packaging Tool (APT) repositories to install software unavailable on the App Store. Cydia is based on APT which was ported to iOS as part of Jay Freeman's Telesphoreo project.
