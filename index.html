What is Cydia

Cydia is a Package Manager application developed by Jay Freeman (aka "Saurik") for Jailbroken iOS devices.

Purpose & Function

Cydia provides a graphical user interface (GUI) to jailbroken users using Advanced Packaging Tool (APT) repositories to install software.

History

Cydia was first released in February 2008 as an alternative to Installer.app on iPhone OS 1.1. In February 2018, Coolstar launched Electra with access to Cydia on iOS 11.0 - 11.1.2.

Cydia Reference

The name "Cydia" is a reference to the Cydia genus of moths, notably the coding moth, which is the proverbial "Worm of the Apple".
